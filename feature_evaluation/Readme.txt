The following folder consists of the main code file: featureSelGameTheory.py and the four data sets used. 
A contribution file GT_contribution.pdf stating the contributions of each team member. 
The file gcmi.py is used as supporting file which contains the functions to calculate mutual information and conditional mutual information. 
To run the code: 
1) copy all the files into the same folder.
2) change the path of the data file in featureSelGameTheory.py in line number 22 to the location where the files are copied to
3) Run the file featureSelGameTheory.py
